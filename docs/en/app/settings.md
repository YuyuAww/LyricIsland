# App Settings

App Settings manage Lyricon's own language, theme, built-in services, and configuration backup.

## Entry

Open Lyricon and enter **Settings**.

## Language

| Option            | Description                              |
|:------------------|:-----------------------------------------|
| Follow system     | Uses the current system language         |
| Specific language | Manually selects the Lyricon UI language |

After changing language, the app refreshes its UI.

## Theme Mode

| Option        | Description                          |
|:--------------|:-------------------------------------|
| Follow system | Follows the system dark mode setting |
| Light         | Forces the light theme               |
| Dark          | Forces the dark theme                |

## Monet Color

On Android 12 and later, Monet color can be enabled. When enabled, the Lyricon app UI follows system
dynamic colors.

This setting affects only the Lyricon app UI. Lyric text colors are configured
in [Text Style](./text-style.md).

## Disable Built-in Services

**Disable built-in services** is useful when using external Lyricon services. When enabled, built-in
lyric services are disabled.

After changing this setting, restart System UI to ensure the service state is fully applied.

## Backup And Restore

The Settings page provides:

| Action  | Description                               |
|:--------|:------------------------------------------|
| Backup  | Exports current Lyricon configuration     |
| Restore | Restores configuration from a backup file |

See [Backup And Restore](./backup-restore.md) for details.

The About page includes version, build time, GitHub link, and open-source license information. When
