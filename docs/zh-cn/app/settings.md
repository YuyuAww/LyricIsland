# 应用设置

应用设置用于管理词幕自身的语言、主题、内置服务和配置备份。

## 入口

打开词幕首页，进入 **设置**。

## 语言

| 选项   | 说明         |
|:-----|:-----------|
| 跟随系统 | 使用系统当前语言   |
| 指定语言 | 手动选择词幕界面语言 |

修改语言后，应用会刷新界面以应用设置。

## 主题模式

| 选项   | 说明           |
|:-----|:-------------|
| 跟随系统 | 根据系统深色模式自动切换 |
| 亮色   | 强制使用亮色主题     |
| 暗色   | 强制使用暗色主题     |

## 莫奈配色

Android 12 及以上设备可以启用莫奈配色。开启后，词幕界面会使用系统动态颜色。

该设置只影响词幕应用界面，不等同于歌词文字颜色。歌词文字颜色请在 [文字样式](./text-style.md) 中配置。

## 禁用内置服务

**禁用内置服务** 适合安装外部词幕服务时使用。启用后，内置歌词服务会被关闭。

修改该设置后，需要重启 System UI 才能确保服务状态完全生效。

## 备份与恢复

设置页面提供：

| 操作 | 说明        |
|:---|:----------|
| 备份 | 导出当前词幕配置  |
| 恢复 | 从备份文件恢复配置 |

详细说明见 [备份与恢复](./backup-restore.md)。
