# 安装与激活

词幕需要通过 LSPosed 或兼容的 Xposed 框架注入 System UI。安装 APK 后，还需要启用模块、配置作用域并重启
System UI。

## 环境要求

| 项目         | 要求                      |
|:-----------|:------------------------|
| Android 版本 | Android 10 (API 29) 及以上 |
| Root       | 需要 Root 环境              |
| LSPosed 框架 | LSPosed 2.0 及更高版本       |
| 作用域        | 必须包含系统界面（System UI）     |

## 安装主体应用

1. 打开 [词幕 Releases](https://github.com/proify/lyricon/releases)。
2. 下载最新版本 APK。
3. 在设备上安装 APK。
4. 打开词幕，确认应用可以正常进入首页。

## 启用 LSPosed 模块

1. 打开 LSPosed 管理器。
2. 找到并启用 **词幕** 模块。
3. 进入模块作用域设置。
4. 勾选 **系统界面 (System UI)**。
5. 保存后重启 System UI 或重启设备。

## 验证激活状态

打开词幕首页，查看顶部状态卡片：

| 状态         | 含义               | 处理方式                         |
|:-----------|:-----------------|:-----------------------------|
| 已激活        | 模块服务可用           | 可以继续安装插件和配置样式                |
| 未激活        | 模块未注入或服务不可用      | 检查 LSPosed 模块和 System UI 作用域 |
| 待重启系统界面    | 词幕更新后需要重新注入      | 点击重启系统界面或手动重启设备              |
| 系统界面异常，已禁用 | System UI 进入保护状态 | 检查 Lsposed 日志和最近修改的配置        |

## 重启 System UI

词幕提供“重启系统界面”入口。该操作通常需要 Root 权限。如果重启失败，请确认 Root 已授权，或直接重启设备。

## 安装歌词提供服务

词幕需要歌词提供服务提供歌词数据。主体应用安装完成后，请根据播放器安装对应插件。

- [LyricProvider Releases](https://github.com/proify/LyricProvider/releases)

安装插件后，重新打开播放器并播放歌曲。若仍无歌词，请查看 [歌词提供服务](./providers.md)
和 [常见问题](./troubleshooting.md)。

## 升级说明

升级词幕后，如果首页提示“待重启系统界面”，说明新版本配置或注入逻辑需要重新加载。此时重启 System UI
或设备即可。
