# 基本设置

基本设置控制歌词容器在状态栏中的位置、尺寸、可见性和过滤策略。它影响所有应用样式，是排查显示位置问题时最先调整的页面。

## 入口

打开词幕首页，进入 **基本设置**。

## 位置与尺寸

| 设置项  | 说明             | 建议                |
|:-----|:---------------|:------------------|
| 锚点   | 选择状态栏中的参考视图    | 歌词位置不准时优先调整       |
| 插入顺序 | 设置歌词插入在锚点之前或之后 | 与锚点组合使用           |
| 宽度   | 限制歌词视图最大宽度     | 避免覆盖时间、信号、电量等系统图标 |
| 边距   | 设置歌词视图外侧留白     | 用于整体位置微调          |
| 内边距  | 设置内容与容器边缘距离    | 用于调整视觉留白          |

边距和内边距的输入顺序为左、上、右、下，单位为像素。

## ColorOS 流体云

在 ColorOS / OPlus 设备上，词幕会提供额外的 **宽度（流体云模式下）** 设置。该设置用于在流体云场景下单独限制歌词视图宽度。

如果进入流体云后歌词挤压或覆盖图标，优先调整该宽度。

## 视图规则

**配置视图可见性规则** 用于控制状态栏中部分视图在播放时的显示方式。常见用途包括让某些系统视图在歌词播放时隐藏，避免重叠。

不同 ROM 的状态栏视图层级差异很大，系统更新后规则可能需要重新配置。

## 歌词屏蔽正则

**歌词屏蔽正则** 用于隐藏匹配正则表达式的歌词行。留空表示不屏蔽。

适合屏蔽以下内容：

- 广告或来源标记。
- 无意义占位文本。
- 不希望显示的固定歌词片段。

## 中文转换

| 选项    | 说明         |
|:------|:-----------|
| 保持原样  | 不转换歌词文本    |
| 转换为简体 | 将中文歌词转换为简体 |
| 转换为繁体 | 将中文歌词转换为繁体 |

## 可见性与隐藏策略

| 设置项       | 说明                      |
|:----------|:------------------------|
| 在锁屏界面隐藏   | 锁屏时不显示歌词                |
| 无歌词隐藏超时   | 没有可用歌词时，经过指定秒数隐藏歌词视图    |
| 无更新隐藏超时   | 歌词停止更新后，经过指定秒数隐藏歌词视图    |
| 触发关键词隐藏超时 | 歌词匹配关键词规则后，经过指定秒数隐藏歌词视图 |
| 关键词正则列表   | 每行一条正则，匹配后触发关键词隐藏超时     |

超时值填 `0` 表示不隐藏。

## 调整建议

1. 先选择锚点和插入顺序，让歌词出现在大致正确的位置。
2. 再调整宽度，避免覆盖系统图标。
3. 最后使用边距和内边距进行细调。
4. 如果只在某些场景重叠，再配置视图规则和隐藏策略。
