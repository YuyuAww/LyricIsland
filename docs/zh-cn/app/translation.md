# 翻译

词幕支持显示翻译歌词。翻译可以来自歌词提供服务，也可以通过 OpenAI 兼容接口生成。

## 翻译来源

| 来源          | 说明                      |
|:------------|:------------------------|
| 歌词提供服务      | 插件直接提供翻译歌词，稳定性取决于插件和播放器 |
| OpenAI 兼容接口 | 词幕调用配置的模型翻译歌词           |

如果插件没有提供翻译，且没有配置 AI 翻译，则不会显示翻译歌词。

## 显示选项

| 设置项    | 说明                |
|:-------|:------------------|
| 仅显示翻译  | 只显示翻译结果，不显示原文     |
| 隐藏翻译   | 不显示翻译歌词           |
| 忽略中文歌词 | 当歌词全部为中文时跳过 AI 翻译 |

## OpenAI 兼容翻译

在 **应用样式** 的 **文字** 标签页中配置 OpenAI 翻译。

| 设置项    | 说明                         |
|:-------|:---------------------------|
| 目标语言   | 翻译输出语言，例如 English、Japanese |
| API 密钥 | 用于访问 OpenAI 兼容服务的密钥        |
| 模型     | 翻译使用的模型名称                  |
| 基础地址   | OpenAI 兼容 API 的 Base URL   |
| 自定义提示词 | 发送给模型的翻译提示词                |

模型设置支持加载可用模型。如果加载失败，请检查 API 密钥、基础地址和网络连接。

## 高级参数

| 参数                | 说明                     |
|:------------------|:-----------------------|
| Temperature       | 控制随机性，越低越稳定，越高越发散      |
| Top P             | 控制 nucleus sampling 范围 |
| 最大 Token 数        | 限制每次翻译生成长度，填 0 使用服务默认值 |
| Presence penalty  | 调整模型引入新表达的倾向           |
| Frequency penalty | 调整模型抑制重复词语的强度          |

不确定如何设置时，建议保持默认值。

## 清空数据库

**清空数据库** 会删除本地缓存的翻译数据。适合以下场景：

- 修改提示词后希望重新翻译。
- 翻译结果异常，想清除缓存。
- 不再需要保留历史翻译。

该操作不可直接撤销，请谨慎使用。

## 隐私和费用

使用 AI 翻译时，歌词文本会发送到你配置的 OpenAI 兼容服务。该服务可能产生费用，也可能记录请求内容。请确认服务提供方的隐私政策和计费规则。

## 常见问题

| 问题        | 处理方式                          |
|:----------|:------------------------------|
| API 密钥未设置 | 填写有效 API 密钥                   |
| 未找到可用模型   | 检查 Base URL 和 API 密钥，或手动填写模型名 |
| 翻译不显示     | 检查是否启用了隐藏翻译，或歌词是否被忽略中文规则跳过    |
| 翻译质量不稳定   | 降低 Temperature，优化自定义提示词       |
