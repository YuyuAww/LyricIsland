# 首次使用

首次使用时，建议按检查清单逐步确认。这样可以先验证模块和歌词来源是否正常，再进行视觉微调。

## 检查清单

1. 词幕 APK 已安装。
2. LSPosed 中已启用词幕模块。
3. 模块作用域已勾选 **系统界面 (System UI)**。
4. System UI 已重启，或设备已重启。
5. 首页模块状态显示为已激活。
6. 已安装当前播放器对应的歌词提供服务。
7. 播放器正在播放有歌词的歌曲。
8. 状态栏出现歌词后，再调整样式。

## 首页状态

首页会显示模块和系统界面的当前状态。遇到问题时，优先根据状态卡片判断原因。

| 状态         | 说明                                 |
|:-----------|:-----------------------------------|
| 已激活        | 词幕模块服务已连接，注入链路正常                   |
| 未激活        | LSPosed 未启用、作用域不正确，或 System UI 未重启 |
| 待重启系统界面    | 当前配置或版本需要重新注入                      |
| 系统界面异常，已禁用 | 为避免影响系统界面，词幕已停止显示                  |

## 第一次测试歌词

1. 打开已支持的音乐播放器。
2. 播放一首确定有歌词的歌曲。
3. 保持播放器在后台或前台均可。
4. 观察状态栏是否出现歌词。
5. 如果没有显示，进入词幕的 **歌词提供服务** 页面确认服务是否可用。

## 第一次调整位置

如果歌词已显示但位置不理想，按下面顺序调整：

1. 在 **基本设置** 中选择合适锚点。
2. 调整插入顺序为之前或之后。
3. 调整宽度，避免覆盖状态栏图标。
4. 使用边距和内边距做小范围微调。
5. 必要时配置视图可见性规则。

## 推荐初始策略

- 先使用默认样式，不急于修改颜色、动画和字体。
- 先确认当前播放器有歌词，再测试其他播放器。
- 如果某个播放器无歌词，优先检查插件支持情况，而不是调整样式。
- 如果歌词覆盖系统图标，优先减小宽度或更换锚点。
