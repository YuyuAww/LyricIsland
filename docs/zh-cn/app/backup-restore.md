# 备份与恢复

备份与恢复用于迁移或保存词幕配置。它适合重装应用、换机、刷机或尝试大量样式配置前使用。

## 备份内容

备份主要保存词幕应用内的配置，例如：

- 基本设置。
- 默认样式和单应用样式。
- 文字、图标、动画配置。
- 翻译相关配置。
- 应用语言、主题等偏好。

> [!Warning]
> 密钥等敏感信息不会被备份。

## 导出备份

1. 打开词幕。
2. 进入 **设置**。
3. 点击 **备份**。
4. 选择保存位置。
5. 保存备份文件。

默认文件名为 `lyricon_backup.bin`。

## 恢复备份

1. 打开词幕。
2. 进入 **设置**。
3. 点击 **恢复**。
4. 选择备份文件。
5. 等待恢复完成。
6. 根据需要重启词幕、System UI 或设备。

## 恢复后检查

恢复完成后建议检查：

1. 首页模块状态是否已激活。
2. 基本设置中的锚点和宽度是否仍适合当前 ROM。
3. 应用样式是否正常显示。
4. 歌词提供服务是否已安装。
5. 翻译 API 配置是否仍可用。

## 跨设备恢复

跨设备恢复时，状态栏结构、屏幕尺寸和 ROM 可能不同。恢复后如果位置异常，优先重新调整锚点、宽度、边距和视图规则。

## 跨版本恢复

从旧版本备份恢复到新版本时，部分新增设置会使用默认值。如果恢复后显示异常，建议重启 System UI 并重新检查关键配置。
