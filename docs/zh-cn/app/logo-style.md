# 图标样式

图标样式控制歌词旁边的 Logo 或封面显示。该页面位于 **应用样式** 的 **图标** 标签页。

## 基础设置

| 设置项  | 说明                   |
|:-----|:---------------------|
| 启用   | 控制是否显示歌词图标           |
| 大小   | 同时设置图标宽度和高度，使图标保持正方形 |
| 边距   | 设置图标外侧留白             |
| 插入顺序 | 控制图标显示在歌词文字之前或之后     |

## ColorOS 设置

在 ColorOS / OPlus 设备上，可以启用 **在流体云模式下隐藏**。开启后，进入流体云场景时会隐藏歌词图标，减少状态栏拥挤。

## 图标来源

| 样式       | 说明                        |
|:---------|:--------------------------|
| 默认       | 使用歌词提供服务或词幕默认图标           |
| 应用图标     | 使用当前播放器应用图标               |
| 音乐封面（方形） | 使用当前歌曲封面，保持方形或圆角形态        |
| 音乐封面（圆形） | 使用当前歌曲封面并裁剪为圆形            |
| 自定义图标    | 使用用户输入的 Base64 图片或 SVG 内容 |

## 自定义图标

选择 **自定义图标** 后，可以点击输入按钮粘贴：

- Base64 编码的图片。
- 原始 SVG 内容。

如果输入内容为空，会取消自定义图标内容。

## 图标颜色

| 设置项    | 说明         |
|:-------|:-----------|
| 自定义颜色  | 启用手动设置图标颜色 |
| 亮色模式颜色 | 亮色模式下图标颜色  |
| 暗色模式颜色 | 暗色模式下图标颜色  |

部分图标来源可能不适合强制着色，例如音乐封面。实际效果取决于图标来源和渲染方式。

## 配置建议

- 状态栏空间紧张时，关闭图标或减小图标大小。
- 希望识别播放器时，使用应用图标。
- 希望视觉更贴合歌曲时，使用音乐封面。
- 使用封面图标时，文字也可开启封面取色，整体更统一。
