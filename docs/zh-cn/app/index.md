# App 使用指南

词幕是基于 Xposed / LSPosed 的 Android 状态栏歌词工具。主体应用负责配置位置、样式、歌词来源和应用偏好，歌词内容会注入到系统界面（System
UI）的状态栏区域显示。

## 适合谁使用

- 希望在状态栏中常驻查看当前播放歌词的用户。
- 希望为不同播放器配置不同歌词样式的用户。
- 需要逐字歌词、翻译歌词、封面取色或状态栏视觉自定义的用户。
- 熟悉 Root、LSPosed 和系统界面作用域配置的 Android 用户。

## 工作方式

词幕由三部分共同工作：

| 组成         | 作用                  |
|:-----------|:--------------------|
| 词幕主体应用     | 管理配置、样式、歌词提供服务和备份恢复 |
| LSPosed 模块 | 将歌词视图注入 System UI   |
| 歌词提供服务     | 从播放器或外部来源提供歌词数据     |

主体应用本身不直接显示状态栏歌词。只有模块在 LSPosed 中启用、作用域包含 System UI，并且 System UI
已重启后，状态栏歌词才会生效。

## 核心功能

| 功能    | 说明                         |
|:------|:---------------------------|
| 状态栏歌词 | 在系统状态栏区域显示当前播放歌词           |
| 逐字歌词  | 支持字词级进度和高亮，具体取决于歌词来源       |
| 翻译歌词  | 支持插件提供翻译，也支持 OpenAI 兼容接口翻译 |
| 应用样式  | 可为不同播放器设置独立文字、图标和动画样式      |
| 位置调整  | 支持锚点、插入顺序、宽度、边距和内边距配置      |
| 图标显示  | 支持默认图标、应用图标、音乐封面和自定义图标     |
| 动画效果  | 支持歌词切换动画预设                 |
| 备份恢复  | 可导出和恢复词幕配置                 |

## 快速开始

1. 从 [Releases](https://github.com/proify/lyricon/releases) 下载并安装词幕。
2. 在 LSPosed 中启用词幕模块。
3. 勾选 **系统界面 (System UI)** 作用域。
4. 重启 System UI 或重启设备。
5. 安装适配当前播放器的歌词提供服务。
6. 打开播放器播放歌曲，确认状态栏出现歌词。
7. 回到词幕调整基本设置和应用样式。

## 使用建议

- 先确认歌词能够正常显示，再调整位置和样式。
- 不同 ROM 的状态栏结构差异较大，锚点和宽度没有通用最佳值。
- 插件是否支持逐字歌词、翻译歌词，取决于歌词来源本身。
- 系统更新、ROM 更新、LSPosed 更新或词幕升级后，如显示异常，优先重启 System UI。
