# 文字样式

文字样式控制歌词文本的大小、颜色、字体、滚动、逐字进度和翻译显示。该页面位于 **应用样式** 的 **文字**
标签页。

## 基础显示

| 设置项         | 说明                 |
|:------------|:-------------------|
| 大小          | 控制歌词文字大小，单位为像素     |
| 边距          | 设置文字区域外侧留白         |
| 内边距         | 设置文字与布局边界之间的留白     |
| 多行模式下文字大小比例 | 多行显示时对文字大小应用的缩放比例  |
| 变换动画速度      | 控制文字布局变化时的动画速度     |
| 边缘淡出长度      | 控制文字边缘淡出效果宽度       |
| 占位内容        | 没有当前歌词时显示歌名、歌手或不显示 |

## 颜色

| 设置项     | 说明            |
|:--------|:--------------|
| 提取封面颜色  | 从当前音乐封面提取文字颜色 |
| 提取封面渐变色 | 从封面生成渐变显示效果   |
| 自定义颜色   | 使用手动设置的文字颜色   |
| 亮色模式颜色  | 亮色模式下使用的颜色    |
| 暗色模式颜色  | 暗色模式下使用的颜色    |

封面取色和自定义颜色通常互斥。开启自定义颜色时，词幕会优先使用手动颜色。

## 字体

| 设置项   | 说明                   |
|:------|:---------------------|
| 自定义字体 | 输入字体路径或名称，留空使用系统默认字体 |
| 字重    | 控制字体粗细，数值越大越粗        |
| 粗体    | 强制使用粗体样式             |
| 斜体    | 强制使用斜体样式             |

如果自定义字体路径错误，系统会回退到可用字体。

## 跑马灯

长歌词超过可显示宽度时，可以使用跑马灯滚动。

| 设置项      | 说明               |
|:---------|:-----------------|
| 跑马灯      | 启用长文本滚动          |
| 速度       | 控制滚动速度           |
| 重复间距     | 设置重复文本之间的间隔      |
| 初始滚动延迟   | 第一次滚动前等待时间，单位为毫秒 |
| 滚动延迟     | 每次滚动循环之间的等待时间    |
| 无限滚动     | 不限制滚动次数          |
| 最大滚动次数   | 未启用无限滚动时最多滚动次数   |
| 结束时在末尾停止 | 滚动结束后停留在歌词末尾     |

## 逐字歌词

逐字歌词依赖歌词提供服务。如果服务没有提供字词级时间轴，词幕只能显示普通整行歌词。

| 设置项      | 说明                   |
|:---------|:---------------------|
| 相对进度     | 歌词缺少字词时间轴时，将整行作为进度单元 |
| 相对进度高亮   | 对整行进度应用高亮效果          |
| 字词动画     | 中日韩歌词按字符动画，拉丁文字按单词动画 |
| 中日韩上浮系数  | 控制中日韩字符动画上浮强度        |
| 中日韩波长系数  | 控制中日韩字符动画波浪间距        |
| 拉丁文字上浮系数 | 控制拉丁文字单词动画上浮强度       |
| 拉丁文字波长系数 | 控制拉丁文字单词动画波浪间距       |

## 配置建议

- 歌词容易遮挡状态栏时，优先减小字号和宽度。
- 长歌词较多时，开启跑马灯比单纯增大宽度更稳定。
- 追求清晰度时，使用固定自定义颜色比封面取色更可控。
- 逐字动画对性能和歌词源要求更高，异常时可先关闭字词动画。
