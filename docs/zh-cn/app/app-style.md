# 应用样式

应用样式用于配置歌词的文字、图标和动画。词幕支持默认样式，也支持为不同应用设置单独样式。

## 默认样式

默认样式会应用到没有单独配置的所有应用。第一次配置时建议先调整默认样式。

## 单应用样式

如果某个播放器需要不同显示效果，可以为它添加单独样式。播放该应用的音乐时，词幕会优先使用它的单独样式。

适合单独配置的场景：

- 某个播放器歌词较长，需要更小字号或更宽显示区域。
- 某个播放器封面取色效果更好，希望单独开启。
- 某个播放器适合显示应用图标，另一个适合显示封面。
- 不同播放器使用不同动画效果。

## 样式优先级

```text
当前播放应用的单独样式 > 默认样式
```

如果没有为当前应用创建样式，则使用默认样式。

## 页面结构

应用样式页面分为三个标签页：

| 标签页 | 说明                   |
|:----|:---------------------|
| 文字  | 字号、颜色、字体、跑马灯、逐字歌词和翻译 |
| 图标  | 图标开关、图标来源、大小、颜色和插入顺序 |
| 动画  | 歌词切换动画开关和预设          |

## 添加应用样式

1. 进入 **应用样式**。
2. 点击标题区域切换或添加应用。
3. 选择需要单独配置的应用。
4. 在文字、图标、动画标签页中修改配置。

## 重置应用样式

如果某个应用样式配置混乱，可以重置该应用样式。重置后，该应用会回到默认样式效果。

## 配置建议

- 先调整默认样式，再为特殊播放器做单独样式。
- 如果只是所有应用都需要变化，不要创建单应用样式。
- 调整样式前先确认基本设置中的位置和宽度已经合适。
