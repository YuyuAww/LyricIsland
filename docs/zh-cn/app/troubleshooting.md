# 常见问题

本页按现象给出排查顺序。遇到问题时，建议先确认首页模块状态，再检查歌词提供服务。

## 歌词完全不显示

按顺序检查：

1. LSPosed 中是否启用词幕模块。
2. 作用域是否勾选 **系统界面 (System UI)**。
3. 是否已经重启 System UI 或设备。
4. 首页模块状态是否为已激活。
5. 是否安装了当前播放器对应的歌词提供服务。
6. 当前歌曲是否有可用歌词。
7. 基本设置中是否配置了过小宽度或隐藏规则。

## 模块未激活

可能原因：

- LSPosed 模块未启用。
- 没有勾选 System UI 作用域。
- System UI 尚未重启。
- LSPosed 或 Root 环境异常。

处理方式：重新检查 LSPosed 配置，并重启 System UI 或设备。

## 提示待重启系统界面

词幕升级或部分设置变化后，需要重新注入 System UI。点击首页的重启入口，或手动重启设备。

如果重启失败，请确认 Root 权限已授予。

## 系统界面异常，已禁用

这是保护机制，用于避免持续影响 System UI。处理方式：

1. 回想最近修改的锚点、视图规则、宽度或样式。
2. 将可疑设置恢复为更保守的值。
3. 重启 System UI。
4. 如仍异常，重启设备后再检查。

## 歌词位置不对

优先调整：

1. 锚点。
2. 插入顺序。
3. 宽度。
4. 边距。
5. 内边距。

不同 ROM 没有通用位置参数，建议小幅调整并反复测试。

## 歌词覆盖系统图标

处理方式：

- 减小基本设置中的宽度。
- 更换锚点。
- 调整插入顺序。
- 关闭图标或减小图标大小。
- 配置视图规则隐藏冲突视图。

## 没有逐字歌词

逐字歌词依赖歌词提供服务。如果插件没有提供字词级时间轴，词幕只能显示普通歌词。

可以检查歌词提供服务页面是否显示 **逐字歌词** 标签。

## 没有翻译

按顺序检查：

1. 歌词提供服务是否支持翻译。
2. 是否开启了隐藏翻译。
3. 是否开启仅显示翻译但没有翻译数据。
4. OpenAI 翻译是否配置 API 密钥、模型和 Base URL。
5. 是否启用了忽略中文歌词。

## AI 翻译失败

常见原因：

- API 密钥错误。
- Base URL 错误。
- 模型名不存在或无权限。
- 网络不可用。
- 服务额度不足。

处理方式：先在翻译设置中加载模型列表，确认服务可访问，再测试实际播放。

## 找不到歌词提供服务

处理方式：

- 确认插件 APK 已安装。
- 确认插件版本与词幕版本兼容。
- 确认词幕拥有读取应用列表权限。
- 重新打开词幕和播放器。
- 必要时重启设备。

## 设置不生效

部分设置需要重新注入 System UI。请尝试：

1. 返回首页查看是否提示待重启。
2. 重启 System UI。
3. 重启播放器。
4. 重启设备。

## 更新后显示异常

更新词幕、ROM、LSPosed 或播放器后，如果出现异常：

1. 重启 System UI 或设备。
2. 检查 LSPosed 作用域。
3. 更新歌词提供服务。
4. 重新调整锚点和宽度。
5. 检查单应用样式是否覆盖默认样式。

## 反馈问题时建议提供

- 词幕版本。
- Android 版本和 ROM 名称。
- LSPosed 版本。
- 播放器名称和版本。
- 歌词提供服务名称和版本。
- 问题截图或录屏。
- 是否使用其他状态栏相关模块。
