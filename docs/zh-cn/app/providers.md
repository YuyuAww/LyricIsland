# 歌词提供服务

歌词提供服务负责把播放器中的歌词、播放状态和翻译等数据提供给词幕。词幕主体应用负责显示配置，不保证所有播放器都能直接提供歌词。

## 服务来源

可以从官方源安装歌词提供服务：

- [LyricProvider Releases](https://github.com/proify/LyricProvider/releases)

## 查看可用服务

进入词幕首页，打开 **歌词提供服务** 页面。页面会按类别显示已安装且可识别的服务。

| 显示项  | 说明                |
|:-----|:------------------|
| 服务名称 | 歌词提供服务或适配模块名称     |
| 作者   | 服务作者信息            |
| 版本   | 当前安装版本            |
| 标签   | 服务支持的能力，例如逐字歌词、翻译 |

## 列表显示模式

歌词提供服务页面右上角可以切换显示模式：

| 模式 | 说明             |
|:---|:---------------|
| 精简 | 只显示核心信息，适合日常查看 |
| 完整 | 显示更多标签和服务信息    |

## 能力标签

| 标签   | 含义                       |
|:-----|:-------------------------|
| 逐字歌词 | 服务可以提供字词级时间轴，用于逐字高亮或字词动画 |
| 翻译   | 服务可以提供翻译歌词               |

没有对应标签时，词幕仍可能显示普通歌词，但不会获得该能力。

## 安装插件后无效

请按顺序检查：

1. 插件 APK 是否已安装成功。
2. 插件是否支持当前播放器版本。
3. 当前播放器是否正在播放有歌词的歌曲。
4. 词幕是否能在歌词提供服务页面看到该服务。
5. 安装或更新插件后，是否重新打开播放器。

## 没有可用服务

如果页面显示没有可用的歌词提供服务：

- 确认已安装对应插件。
- 确认词幕拥有读取应用列表所需权限。
- 确认插件包名和版本与当前词幕兼容。
- 尝试重新打开词幕和播放器。
